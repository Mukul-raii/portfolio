## Overview

**Miss-Minutes** is a sophisticated automated coding analytics platform that seamlessly integrates with Visual Studio Code to track your coding time automatically. Unlike traditional time tracking tools that require manual intervention, Miss-Minutes works silently in the background, capturing every keystroke, file edit, and commit with millisecond precision. The platform combines a powerful VS Code extension with a modern web dashboard to provide developers with comprehensive insights into their coding habits and productivity patterns.

## Key Features

### **Automatic Time Tracking**

- **Zero Configuration**: Start coding and Miss-Minutes tracks everything automatically without any manual setup
- **File-Level Precision**: Tracks every file you edit with millisecond accuracy for detailed insights
- **Git Integration**: Automatically links coding activities to commits and branches for comprehensive version control tracking
- **Multi-Project Support**: Seamlessly switch between projects without manual intervention or configuration

### **Powerful Analytics Dashboard**

- **7-Day Activity Overview**: Visual charts and graphs showing your coding patterns over the past week
- **Language Breakdown**: Comprehensive analysis of which programming languages you use most frequently
- **Project Insights**: Detailed statistics for each project including time spent, commits, and activity patterns
- **Real-time Updates**: Live tracking with automatic synchronization between extension and dashboard
- **Heatmap Visualization**: Interactive heatmaps to understand your productivity patterns throughout the day

### **Privacy First Approach**

- **Local Storage**: Extension caches data locally using SQLite for offline support and privacy
- **Secure Sync**: API token-based authentication ensures secure cloud synchronization
- **No Source Code Tracking**: Only metadata is tracked (file names, languages, duration) - never your actual code
- **You Own Your Data**: Full control over your coding activity data with export capabilities

### **Beautiful User Interface**

- **Modern Design**: Clean, responsive interface built with Next.js 14 and Tailwind CSS
- **Dark/Light Mode**: Automatic theme switching based on system preferences
- **Mobile Responsive**: Access your dashboard from any device with fully responsive design
- **Intuitive Navigation**: Easy-to-use interface with clear data visualizations

## System Architecture

### **Data Flow**

```mermaid
graph LR
    A[VS Code Extension] -->|Tracks Activity| B[Local SQLite DB]
    B -->|Syncs Every 5 Min| C[GraphQL API]
    C -->|Stores Data| D[PostgreSQL]
    D -->|Queries| E[Next.js Dashboard]
    E -->|Displays| F[Analytics & Charts]
```

1. **Extension monitors** file changes, cursor activity, and Git events in real-time
2. **Local caching** stores activities in SQLite for offline support and performance
3. **Background sync** sends aggregated data to the cloud API every 5 minutes
4. **Dashboard queries** the PostgreSQL database for analytics and visualizations
5. **Real-time updates** show your current coding session live on the dashboard

## API Architecture

### **GraphQL Schema**

```graphql
# Core GraphQL types
type Activity {
  id: ID!
  userId: String!
  projectId: String!
  filePath: String!
  language: String!
  duration: Int!
  timestamp: DateTime!
}
type Project {
  id: ID!
  name: String!
  path: String!
  activities: [Activity!]!
  commits: [Commit!]!
  totalDuration: Int!
  languages: [LanguageStats!]!
}
# Main queries
type Query {
  activities(dateRange: DateRange!): [Activity!]!
  projects: [Project!]!
  analytics(projectId: ID): AnalyticsSummary!
}
# Mutations for data sync
type Mutation {
  syncActivities(activities: [ActivityInput!]!): SyncResult!
  createProject(name: String!, path: String!): Project!
}
```
